# Getting more out of a computer

Sometimes, a computer application doesn't quite do what I want it to do, so I have to ask the computer nicely, in a different way. Sometimes, that means opening up the Developer Tools in the browser and running a script on the console to add a missing feature to a website even if just for the current session. Other times, it means editing system files, changing source code and recompiling, or throwing AI at the problem.
